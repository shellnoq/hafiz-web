# Changelog

## [0.1.0] - 2024-12-08

### Added

- Full S3 API compatibility (76+ endpoints)
- Multi-part uploads
- Object versioning
- Lifecycle policies
- Server-side encryption (AES-256-GCM)
- Object Lock (WORM)
- Bucket policies
- LDAP integration
- Admin API & UI
- Prometheus metrics
- Event notifications
- Cluster mode
- Helm chart
- CLI tool

### Technical

- 33,000+ lines of Rust
- 9 crates
- PostgreSQL & SQLite support
